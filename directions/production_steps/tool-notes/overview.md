# Notes App Master Spec + Cursor Build Prompt

## Product concept
Build a full-scale modern notes application that merges the strongest patterns from the leading notes products on the market into one system.

Working name: **Atlas Notes**
Positioning: **Fast enough for capture, powerful enough for knowledge work, structured enough for teams, private enough for serious users.**

The product should feel:
- As easy to start as Apple Notes / Google Keep
- As structured as Notion / OneNote
- As connected as Obsidian / Capacities / Anytype
- As privacy-aware as Joplin
- As polished as Bear

---

# 1. Product thesis
Most note apps force users to choose one of the following:
- speed vs structure
- simplicity vs power
- cloud convenience vs privacy
- documents vs databases vs linked knowledge
- personal note-taking vs collaborative workspace

This app should remove that tradeoff.

The system should support five core modes inside one product:
1. Quick Capture
2. Rich Notes
3. Connected Knowledge
4. Structured Collections
5. Team Collaboration
